# 纽约随手账

一个面向纽约华人使用习惯的轻量记账工具，适合在 iPhone Safari 中打开，也可以添加到主屏幕当作简易 App 使用。

## 这次重设计后的重点

- 移动端优先，底部导航更适合单手操作
- 记账时直接记录时间、收支、金额、分类、支付方式
- 分类更贴近纽约华人常见场景：房租水电、地铁通勤、打车、外卖堂食、买菜华超、红包往来、副业等
- 自动生成本月开源节流建议，帮助你看清哪些地方最值得优先调整
- 数据默认只保存在浏览器本地 `localStorage`

## 如何使用

这是一个纯前端静态项目，不需要后端。

### 直接打开

直接打开 `index.html` 即可使用。

### 本地静态服务

如果你本机装了 Python，可以在项目目录运行：

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://localhost:4173
```

## 在 iPhone 上使用

1. 用 Safari 打开这个页面。
2. 点击“分享”。
3. 选择“添加到主屏幕”。
4. 之后就可以像 App 一样全屏打开，随时记账。

## 当前范围

- 本地版，无登录
- 无云同步
- 无银行自动导入
